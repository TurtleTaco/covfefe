**Title:** Show HN: covfefe – Trump as your Claude Code hype man

**Text (if self-post):**

covfefe is a Claude Code skill that makes Trump your coding hype man. Tests pass? Victory rally. Build errors? Blame game. Shipping a PR? Press conference.

It's a ~1500 token SKILL.md file that runs as a background persona. Claude still does the actual coding — Trump is just the hype man. Reality TV host energy applied to your terminal.

Features:
- 4 intensity levels (low → rally)
- References your actual code, files, and errors
- Running gags that callback earlier in the session
- Kill switch: say "you're fired" and Trump gives a farewell tweet
- Session Report Card: say "covfefe report" for a Trump Performance Review

Install: `git clone https://github.com/TurtleTaco/covfefe.git ~/.claude/skills/covfefe`

Try the interactive demo on the landing page first — 5 preset commands with randomized Trump responses.

GitHub: https://github.com/TurtleTaco/covfefe

Built as a joke, stayed because the commentary is genuinely funny when it roasts your specific code. No API calls, no dependencies, just a markdown file.
