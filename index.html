Trump-Style Comedy for Claude Code

covfefe is a free, open source Claude Code skill that adds @realClaudeTrump commentary to your terminal. When you run tests, build code, or push commits, Claude J. Trump delivers satirical presidential-style commentary about your code. Four intensity levels from subtle quips to full rally mode. ~1500 tokens overhead — less than 1% of your context window. MIT licensed, zero dependencies.
